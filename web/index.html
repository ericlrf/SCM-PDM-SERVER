API Taxas de Câmbio da moeda brasileira

REST service

JSON data

URL: http://localhost:8084/ServidorPDM/TaxasCambio?moeda=K&operacao=X

K = informar o símbolo da moeda de um país (de acordo com a ISO-4217)

X = informar o identificador da função desejada (consultar a lista de funções)

Exemplo

URL: http://127.0.0.1:8084/ServidorPDM/TaxasCambio?moeda=usd&operacao=cotacao
